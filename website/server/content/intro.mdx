# 项目介绍

## 概述

本项目是一个基于现代Web技术栈构建的全栈应用，提供文件处理、AI聊天和性能监控等功能。

## 主要功能

### 文件处理

- 大文件分片上传
- 断点续传
- 秒传验证
- 文件哈希计算

### AI聊天

- 支持OpenAI和百炼API
- 流式响应
- 历史对话记录
- 自定义提示词

### 性能监控

- 内存使用监控
- 页面性能指标
- Jank/Stutter检测
- 虚拟滚动优化

## 技术栈

### 前端

- React 18
- TypeScript 5
- Vite
- Antd 5

### 后端

- Node.js
- Koa 2
- TypeScript
- Swagger UI

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## API文档

访问 `/swagger` 查看完整API文档。
